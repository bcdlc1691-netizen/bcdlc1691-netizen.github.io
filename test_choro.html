Assignment 1: Choropleth map - interactive Leaflet

SDG 5.5.1 interactive map

SDG 5:

Acheive gender equality and empower all women and girls.

SDG Target 5.5

Ensure women's full and effective participation and equal opportunities for leadership at all levels of decision making.


Even though this statistic is not fully indicative of gender equality and the progress towards SDG 5, it is relative value that can help us see patterns and areas of interest. This dataset was obtained from the UN open source Geodata, and it dates to 2020. For this assignment we completed 2 maps, a static print layout and an interactive Leaflet map.






Map aim

The map is showing the percentage of seats in parliament held by women in every country, using a Mercator projection, and divided into 5 color categories, in chunks of 10%, and no data. Additionally, 3 countries with more than 50% are identified and labelled. The title introduces the SDG and its target. This is 5.5.1, more information can be found here. https://unstats.un.org/sdgs/metadata/files/Metadata-05-05-01a.pdf

Why choropleth?

Benefits: Easy visualization, quick to understand, conveys 1 variable effectively
Drawbacks: Generalizes by nation-state, can be difficult to read for colorblind people

First time using QGIS

Using QGIS was completely new so this was an introduction to the software. I also learnt the different file types needed to work with GIS and how to manage them basically in the program. I had a lot of trouble with countries disappearing and turns out it was because when joining both layers, only integers were processed and numbers with decimals were classified as null. That set me back quite a bit until I figured out what was the error, and in that time I redid the map a couple times and got more familiar with QGIS.
